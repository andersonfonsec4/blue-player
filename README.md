# BluePlayer

BluePlayer é um player de música feito em JavaScript puro.

## Recursos

- Upload de músicas locais
- Playlist dinâmica
- Drag & Drop para reorganizar músicas
- Equalizador com Web Audio API
- Presets de equalização
- Visualizer de áudio
- Barra de progresso
- Controle de volume
- Fullscreen
- PWA instalável

## Tecnologias

- HTML
- CSS
- JavaScript
- Web Audio API
- Canvas
- Service Worker (PWA)

## Como usar

1. Abra o `index.html`
2. Clique em **Adicionar músicas**
3. Selecione arquivos de áudio
4. Aproveite o player
